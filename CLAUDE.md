# learning-lab

技術学習のメモと練習コードをまとめるリポジトリ。

## ワークフロー

### 新しいトピックの学習

1. ユーザーがトピックを伝える
2. トピックの概要を説明する
3. 一緒にコードを書いて動作を確認する
4. `templates/topic.md` をベースにノートを作成する

### コードレビュー

- ユーザーが書いたコードの改善点を指摘する
- なぜその改善が必要かを説明する

### 疑問の深掘り

- 「なぜ？」を繰り返して理解を深める
- 表面的な回答で終わらず、内部の仕組みまで掘り下げる

## ファイル規則

### ノート

- パス: `<言語>/notes/<トピック名>.md`
- 例: `go/notes/goroutine.md`
- テンプレート: `templates/topic.md` を使用する

### 練習コード

- パス: `<言語>/code/` 以下に配置
- ノートからコードファイルへのリンクを貼る

## ディレクトリ構成

```
<言語>/
├── notes/    # 学習メモ（Markdown）
└── code/     # 練習コード
```

新しい言語・トピックを追加する場合はこの構成に従う。
